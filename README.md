# 🌐 Portfolio Website

A modern and responsive portfolio website built using **React.js** to showcase my skills, projects, and experience as a Full Stack Developer.

## 🚀 Features

- Responsive design
- Modern and clean UI
- About Me section
- Skills section
- Projects showcase
- Contact section
- Smooth navigation
- Fast performance

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- Tailwind CSS
- React Icons

## 📦 Installation

```bash
git clone https://github.com/pashu111/Ashutosh-Portfolio.git

cd portfolio

npm install

npm run dev
```
